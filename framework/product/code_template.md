# Assignment as a Code

## Ментальная модель

```yaml
${сущность}: !model
  определение: …
  примеры:
    ${название}: …
  расширения:
    ${название}: …
```

## Функция

```yaml
${роль} — ${работа}: !func
  ${название истории}: !story
    сделано: yes
    ситуация: …
    успех: …
    отказ: …
    что понадобится:
      - ${продукт}. ${функция}
  # … ещё истории …
```
