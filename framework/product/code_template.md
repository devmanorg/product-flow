# Assignment as a Code

## Концепты

```yaml
${сущность}: !concept
  определение: …
  ${пример}: !example |
    ${описание}
```

Дополнительные разновидности примеров:
  - `!extension`
  - `!exclusion`

## Роль

```yaml
${название}: !role
  мотивация: …
  старт: …
  ${пример}: !example |
    ${описание}
```

Дополнительные разновидности примеров:
  - `!extension`
  - `!exclusion`

## Функция

```yaml
${роль} — ${работа}: !func
  ${название истории}: !story
    сделано: yes
    ситуация: …
    успех: …
    отказ: …
    что понадобится:
      - ${продукт}. ${функция}
  # … ещё истории …
```

Другая разновидность функции: `!func-export`.

Дополнительные разновидности историй:
  - `!extension`
  - `!exclusion`

## Импорт

```yaml
${новое название функции}: !import
  from: ${rel_path_or_url}
  func: ${название функции}
```
